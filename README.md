
FamilyPod
An innovative platform providing tools for social workers to complete assessments and standardise social care nationwide in the UK. 

Description
FamilyPod is designed to support social workers by streamlining assessments and offering interactive tools to help families navigate challenges. The MVP includes features like therapeutic games such as Romance Roulette and Trust Tower, which aim to teach young people about healthy relationships.
